# 面试过程

## 面试态度
在就是面试过程，尽量保持一些微笑，然后有机会(在没有话题的时候或者啥)就自己主动问面试官的一些问题，不是技术问题。就是扯一下他们是做什么的，然后一些公司的业务啥的。为了让你主导一下，而不是一味的问你，尽量保持愉悦。保持微笑愉悦，自然舒适，让人感觉和你好沟通。
